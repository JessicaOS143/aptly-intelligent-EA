# Aptly Intelligent & Co.

Personal website for Jessica Ocasio Salters — Boutique Executive Operations for Visionary Leaders.

## Built with
- React 18
- Vite
- Hosted on Vercel / Netlify

## Run locally
```bash
npm install
npm run dev
```

## Deploy
Connect this repo to [Vercel](https://vercel.com) or [Netlify](https://netlify.com) for automatic deployment on every push.
